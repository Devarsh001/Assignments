  • Source

    • 2. How to embed audio and video in a webpage?

    We can embed audio and video in a web page using audio tag (<audio>) and video tag (<video>).

    To get the audio or video there are two ways to get audio or video to webpage after audio and video tag.

  • Using src attribute inside the starting audio tag or video tag
  • Using source tag inside the audio tag or video tag

    • 3. Semantic element in HTML5?

    Semantic elements are the elements that describes the meaning of its own to browser as well as to the developer.

    For example; form tag (<form>) - we can clearly say that this tag is used to create form in webpage, also its meaning is same for browser also.

    Examples of semantic elements are; form, table, header, footer, section, aside ...

    4. Canvas and SVG tags

    Canvas:

    HTML canvas element is used to draw graphics on webpage.

    The canvas element is only a container for graphics. You must use JavaScript to actually draw the graphics.

    SVG:

    SVG stands for scalable Veltor Graphics.

    SVG is an XML-based image format that is used to define two-dimensional vector-based graphics for the web.

    SVG images can be created and edited with any text editor.

    SVG image can be scaled up or down to any extent without losing the image quality.

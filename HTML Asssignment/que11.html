Block-level elements occupies the 100% width of the body in HTML.

Some blocklevel elements are listed below:

  • all heading tags (h1 to h6)
  • paragraph tag
  • hr
  • List tags
  • blockquote
  • form tag
  • table tag
  • etc...

    • Inline elements occupies only space tha its content has occupied.

    For example, if an inline element has only 4-5 words of content ; then the element will occupy only that much amount of space in HTML body.

    Some inline elements are listed below:

  • image tag
  • anchor tag
  • bold tag
  • italic tag
  • input tag
  • span tag
  • etc..

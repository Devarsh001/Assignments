In HTML table; if we want to merge (club) two or more rows in single row of table, we use the rowspan attribute.

Example: <tr rowspan="numbers of rows we want to merge into a single row"> </tr>

Similarly, in HTML table; if we want to merge (club) two or more columns in single column of table, we use the colspan attribute.

Example: <tr> <td colspan="numbers of columns we want to merge into a single lolumn"> </td></tr>

The differences between HTML and XHTML:

HTML XHTML
HTML stands for Hyper Text Mark-up Langyage. XHTML stands for Extensible Hypertext Mark-up Language.
HTML is case not sensitive. XHTML is case sensitive.
HTML ignores small errors. XHTML does not allow errors.
In HTML, it is not necessary to mention quotes for attributes. In XHTML, it is mandatory to mention quotes for attributes.
In HTML it is not nessasary to close every tags. In XHTML every tag should be closed.
